# Data Notes

Raw data is not included. Required columns:

| Column | Type | Description |
|---|---|---|
| `post_id` | str | Unique post ID |
| `text` | str | Post or article text |
| `label` | int | 1 = disinformation, 0 = credible |
| `source` | str | Platform or outlet |
| `date` | date | Publication date |

## Suggested Datasets
- LIAR dataset (politifact)
- FakeNewsNet
- Your own manually coded dataset
